จำเป็นหน้าบทความ GEO

คอนเทนต์ที่เผยแพร่ = หน้าจริง (เนื้อหา + schema ในตัว) ที่ platform โฮสต์ให้ — เพิ่ม proxy path เดียวให้ขึ้นใต้โดเมนคุณ ไม่ต้องเขียนโค้ด

Nginx — เพิ่มใน server block แล้ว reload (Plesk/cPanel วางช่อง "Additional nginx directives")
location /geo/ {{ '{' }}
    proxy_pass {{ embed_articles_url }};
    proxy_set_header Host {{ embed_host }};
    proxy_ssl_server_name on;
{{ '}' }}
เสร็จแล้วบทความจะอยู่ที่ โดเมนคุณ/geo/… — กด "เผยแพร่ (public)" ในหน้าคอนเทนต์เพื่อให้ขึ้น
ทางอื่น (Apache / ลิงก์ตรง / ดึง JSON เอง)
Apache: ProxyPass /geo/ {{ embed_articles_url }}
ลิงก์ตรง (ไม่ proxy): {{ embed_articles_url }}
render เองในสแตกตัวเอง: ดึงฟีด {{ embed_content_json_url }}
